Dzień dobry. Napisałem aplikację śledzącą nawyki (habit tracker) - dużą część wymagań pokrywa backend w pythonie, ale dla większej czytelności stworzyłem folder oop_examples, w którym znajdują się skrypty z wymaganym kodem.
Backend jest we Flasku z bazą danych w SQLite, frontend vanilla JS, HTML i CSS (pisany w scss).
